# Ivoc UI repository

`Ivoc` or I Vocabulary - is a platform where you can effortlessly expand your vocabulary. Create an account and start adding words you want to learn. Each word can be categorized as important, less important, or not important.

Build custom collections of words and set up automatic flashcards delivered to your Telegram channel. Choose how many words you want to see and how often. Forget about rote learning â€“ just see the words regularly in your Telegram feed.


## How to start the project?
- Clone repo to your PC
- Ask `.env` file
- Run `npm run dev` in root folder
